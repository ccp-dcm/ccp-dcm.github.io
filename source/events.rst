Hackathon: Data assimiliation in Firedrake
------------------------------------------

A hackathon on variational data assimilation in Firedrake will be held at
Rutherford Appleton Laboratory on 8-10 April 2026. This is an opportunity for
users with an interest in data assimilation to try out the latest high
productivity software technology in this field. More information is on `the
Hackathon website
<https://www.numerical.rl.ac.uk/events/hackathon-data-assimilation-in-firedrake/>`__.
